Conócenos

Somos una organización que quiere incentivar el uso de semillas para que las personas puedan tener huertas en sus hogares

Nuestro producto consiste en entregar cajas de suscripción mensual, las cuales tienen diferentes tipos de semillas que podrán cultivar en sus hogares, se les enviará un manual con instrucciones para realizar el cultivo.

Para los clientes que lo requieran existe un sistema de monitoreo de los nutrientes y condiciones ambientales presentes en el cultivo, lo cual se hará con el uso de sensores, cuyos datos tomados se pueden visualizar en una página web desde el celular
